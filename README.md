# react-native-adhan

**react-native-adhan** is a high-performance native module for React Native that calculates accurate Islamic prayer times using the official [adhan-cpp](https://github.com/batoulapps/adhan-cpp) library. Designed for speed, reliability, and mobile performance, this module works seamlessly on both iOS and Android and is ideal for prayer apps, masjid tools, or Islamic productivity tools.✅ Powered by C++ for fast bulk calculations  � Lightweight & easy to integrate  �� Perfect for mobile prayer tracking and performance logs�

## Installation


```sh
npm install react-native-adhan
```


## Usage


```js
import { multiply } from 'react-native-adhan';

// ...

const result = multiply(3, 7);
```
